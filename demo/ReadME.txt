1. Download & Install https://downloadarchive.documentfoundation.org/libreoffice/old/4.2.4.2/win/x86/LibreOffice_4.2.4.2_Win_x86.msi.
2. open My_computer->C:\Program Files (x86)\LibreOffice 4\program , copy the path
3. go to environment setting
4. click on system_variables->Path (Double click on it and past copyed path) click on apply
5. goto convert.php and edit doc path
6. run convert.php

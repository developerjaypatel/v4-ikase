MySQL Workbench Instructions

1. Create ikase_nameGoeshere database (ex: ikase_phan)

2. Open ikase_basictemplate. This schema has all the up-to-date tables, without data.

3. Select all the Tables in ikase_basictemplate

4. Right Click, Send to SQL Editor | Create Statement

5. Right Click on ikase_nameGoeshere, Set as Default Schema

6. Run the Create Statements

 

StarLinkCMS Customer Management Instructions

1. Login as Master Admin

2. Look up/Create Customer, click on Customer Name to edit customer information

3. Check the Import Permissions box and Save

4. in the Data field, enter ikase_basictemplate (ex: phan)

5. If importing, select from Legacy drop down

6. Save. Click on Import link on upper right. If no legacy, it will be called Basic Import

7. If Basic Import, click all the links one by one, and you're done

8. If legacy, do all the links on the left, then on the right. Let Nick do those for now, as they will need "massaging", because the data tends to have errors in it.
